Steps to set up an Angular app

  • bower init, bower install angular --save
  • don't forget to create a .gitignore and ignore your dependencies
  • In your javascript:
    • create your module
    • set up your config
    • create controller(s)
    • create factory/ies -- optional
    • create directive(s) -- optional
  • In your HTML:
    • add all appropriate script tags
    • ng-app="nameOfApp"
    • ng-view
    • create partial(s)
    • create directive(s) -- optional
